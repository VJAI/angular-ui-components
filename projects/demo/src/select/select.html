Select

Overview

The Select component helps to pick single or multiple from group of options.

Data Binding

You can bind data using ngModel or reactive directives.

Bangalore Goa Chennai 
<lego-select label="Favourite Destination" required [(ngModel)]="selectedCity">
  <lego-select-option key="1">Bangalore</lego-select-option>
  <lego-select-option key="2">Goa</lego-select-option>
  <lego-select-option key="3">Chennai</lego-select-option>
</lego-select>

Required Validation

You can make a dropdown mandatory by adding the required attribute to it.

Male Female Other 
<lego-select label="Gender" required [(ngModel)]="selectedGender">
  <lego-select-option>Male</lego-select-option>
  <lego-select-option>Female</lego-select-option>
  <lego-select-option>Other</lego-select-option>
</lego-select>

Disabled

Adding the disabled attribute disables the dropdown.

Pizza Pasta Ice Cream 
<lego-select label="Snack" placeholder="Pick Your Snack" required disabled [(ngModel)]="selectedSnack">
  <lego-select-option>Pizza</lego-select-option>
  <lego-select-option>Pasta</lego-select-option>
  <lego-select-option>Ice Cream</lego-select-option>
</lego-select>

Empty Options

You can customize the message that is displayed when there are no options exists using the empty-text attribute.

</lego-select>

</lego-select>

Dropdown Widths

As default the width of the dropdown is in sync with the width of the reference input. You can turn off that behavior by passing sync-width as false.

Pizza Pasta Ice Cream 
<lego-select label="Snack" placeholder="Pick Your Snack" [sync-width]="false">
  <lego-select-option>Pizza</lego-select-option>
  <lego-select-option>Pasta</lego-select-option>
  <lego-select-option>Ice Cream</lego-select-option>
</lego-select>

You can set the width of dropdown or control the min and max widths by passing custom css class as shown in the below example.

Pizza Pasta Ice Cream 
<lego-select label="Snack" placeholder="Pick Your Snack" [sync-width]="false" dropdown-css="custom-width">
  <lego-select-option>Pizza</lego-select-option>
  <lego-select-option>Pasta</lego-select-option>
  <lego-select-option>Ice Cream</lego-select-option>
</lego-select>

.custom-width { width: 200px; }

When sync-width is true you can control the min and max widths by passing values to min-dropdown-width and max-dropdown-width attributes.

Pizza Pasta Ice Cream 
<lego-select label="Snack" placeholder="Pick Your Snack" [max-dropdown-width]="200">
  <lego-select-option>Pizza</lego-select-option>
  <lego-select-option>Pasta</lego-select-option>
  <lego-select-option>Ice Cream</lego-select-option>
</lego-select>

Multiple

Like the native element, specifying the multiple attribute will turn the dropdown to multi-selectable.

Superman Batman Hulk Spiderman Catwoman Thor Captain America Ironman 
<lego-select label="Superheros" placeholder="Select Your Superheros" multiple>
  <lego-select-option>Superman</lego-select-option>
  <lego-select-option>Batman</lego-select-option>
  <lego-select-option>Hulk</lego-select-option>
  <lego-select-option>Spiderman</lego-select-option>
  <lego-select-option>Catwoman</lego-select-option>
  <lego-select-option>Thor</lego-select-option>
  <lego-select-option>Captain America</lego-select-option>
  <lego-select-option>Ironman</lego-select-option>
</lego-select>

Change Event

The select component fires change event whenever the selection changes.

Pizza Pasta Ice Cream 
<lego-select label="Snack" placeholder="Pick Your Snack" (change)="onChangeHandler()">
  <lego-select-option>Pizza</lego-select-option>
  <lego-select-option>Pasta</lego-select-option>
  <lego-select-option>Ice Cream</lego-select-option>
</lego-select>
