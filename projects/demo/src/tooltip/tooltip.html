Tooltip

Overview

Tooltips helps to display additional information. You can add tooltip to any element through the directive.

Example

You can enable tooltip on a component or element by passing text to lego-tooltip attribute.

Know More 
<lego-button lego-tooltip="l.e.g.o is the best UI library ever made">>Know More</lego-button>

Dark Mode

You can enable dark mode by passing dark-mode attribute.

Know More 
<lego-button type="secondary" lego-tooltip="l.e.g.o is the best UI library ever made">>Know More</lego-button>

Maxlength

You can control the mexlength of the tooltip text by passing value to the tooltip-length attribute.

Know More 
<lego-button lego-tooltip="This is a lengthier text. This is a lengthier text. This is a lengthier text. This is a lengthier text. This is a lengthier text." [tooltip-length]="50">Know More</lego-button>
