Chips

The chips component can used to display a collection like selected items in a picklist.

Example

HTML CSS JavaScript 
<lego-chips>
  <lego-chip>HTML</lego-chip>
  <lego-chip>CSS</lego-chip>
  <lego-chip>JavaScript</lego-chip>
</lego-chips>
