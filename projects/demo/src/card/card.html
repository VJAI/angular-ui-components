Card

Overview

Cards are used to display compact information of a record. Lego card adds a nice box shadow to the content.

Single Card

Captain America

 
<lego-card>
  <demo-card-content>
    <div class="header">
      <img src="./assets/images/Captain-America.jpg" width="400">
    </div>
    <div class="footer">
      <h4>Captain America</h4>
    </div>
  </demo-card-content>
</lego-card>

List Of Cards

Lego provides a list component that helps to display collection of cards in a single row or as a matrix. Using the spacing property you can control the spacing between the card.

Single Row

As default the cards are displayed in a single row.
