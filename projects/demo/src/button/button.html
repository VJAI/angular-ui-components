Buttons

Overview

Button is one of the primary component required for any decent application. Lego supports three types of buttons.

  • Primary
  • Secondary
  • Tertiary

Button Types

    
<lego-button label="Primary Button"></lego-button>
<lego-button label="Secondary Button" type="secondary"></lego-button>
<lego-button label="Tertiary Button" type="tertiary"></lego-button>

Disabled Button

<lego-button label="Disabled Button" disabled></lego-button>

Listening to "click" event

You can wire-up custom handler to button "click" event using the click binding. 

<lego-button
  label="Verify Account"
  type="secondary"
  (click)="verifyAccount()">
</lego-button>
