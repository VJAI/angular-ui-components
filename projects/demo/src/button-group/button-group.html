Button Group

Overview

Button Group component allows you to group a series of options together (on a single line) in a button group. Only one can be selected at the same time.

Demo

<lego-button-group selected="2" label="Select Phone">
  <lego-button-group-option key="1" value="iPhone">
  </lego-button-group-option>
  <lego-button-group-option key="2" value="Samsung">
  </lego-button-group-option>
  <lego-button-group-option key="3" value="Motorola">
  </lego-button-group-option>
</lego-button-group>

Disabled

<lego-button-group disabled selected="Samsung">
  <lego-button-group-option value="iPhone">
  </lego-button-group-option>
  <lego-button-group-option value="Samsung">
  </lego-button-group-option>
  <lego-button-group-option value="Motorola">
  </lego-button-group-option>
</lego-button-group>

Using With Reactive Form

<lego-button-group [(ngModel)]="selectedPhone" label="Select Phone">
  <lego-button-group-option key="1" value="iPhone">
  </lego-button-group-option>
  <lego-button-group-option key="2" value="Samsung">
  </lego-button-group-option>
  <lego-button-group-option key="3" value="Motorola">
  </lego-button-group-option>
</lego-button-group>
