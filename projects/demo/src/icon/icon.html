Iconography

The Lego icon library is a set of SVG icons that can be used to add inline SVG to html. This allows for developers to easily size and color individual icons within their application.

Basic Icon

The basic usage of the <lego-icon> component will produce a 24 X 24 inline svg icon that inherits the color attribute of it's parent. 

<lego-icon icon="Nav_Cog"></lego-icon>

Custom Styling

Icons can accept styling via CSS to customize properties such as color and size. The default size will be 24 X 24 and the color will be the current text color of the parent element. Styling can also be applied to the css pseudo classes such as hover and active.

Custom Coloring

To override the color of the icon simply set the css color property to the desired value. 

<lego-icon icon="Nav_Bell" style="color: red"></lego-icon>
<lego-icon icon="Nav_Bell" style="color: green"></lego-icon>
<lego-icon icon="Nav_Bell" style="color: blue"></lego-icon>
<lego-icon icon="Attachment" style="color: red"></lego-icon>
<lego-icon icon="Attachment" style="color: green"></lego-icon>
<lego-icon icon="Attachment" style="color: blue"></lego-icon>

Custom Sizing

To override the size of the icon set both the width and height of the icon. 

<lego-icon icon="Customers_Group" style="width: 36px; height: 36px;"></lego-icon>
<lego-icon icon="Customers_Group" style="width: 48px; height: 48px;"></lego-icon>
<lego-icon icon="Customers_Group" style="width: 64px; height: 64px;"></lego-icon>

Hover Effect

HTML

<lego-icon id="moreInfo" icon="Info_Circle"></lego-icon>

CSS

#moreInfo {
 cursor: pointer;
 transition: color .2s ease-in-out;
}

#moreInfo:hover {
 color: #0079CC;
}

Icons can be given special styling through the css pseudo class hover. This is useful for clickable icons used for controlls.
