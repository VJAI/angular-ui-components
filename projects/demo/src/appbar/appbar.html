AppBar

Overview

The AppBar component provides a responsive site navigation header that displays logo, navigation links, profile image and more.

Example

The below code shows the sample appbar. 

<lego-appbar
  logo="./assets/images/microsoft.png"
  home-url="https://microsoft.com">
  <lego-menu-left-item display="Posts"></lego-menu-left-item>
  <lego-menu-left-item display="Tags"></lego-menu-left-item>
  <lego-menu-left-item display="Users"></lego-menu-left-item>
  <lego-menu-right-item>
    <lego-appbar-profile profile-img="./assets/images/profile.png"></lego-appbar-profile>
  </lego-menu-right-item>
</lego-appbar>
