Typography

Introduction

Lego uses Roboto font as the primary choice for both body text and headings. The base font size is set to 14px and the body text color is #323232.

Headings

Four level of headings are supported from h1 — h4. You can also decorate anything into a heading by applying classes lego-h1 — lego-h4.

This is heading <h1>

This is heading <h2>

This is heading <h3>

This is heading <h4>

 
<h1>This is heading <h1></h1>
<h2>This is heading <h2></h2>
<h3>This is heading <h3></h3>
<h4>This is heading <h4></h4>

Paragraphs

You can transform anything into a paragraph by applying lego-body class.

Hyperlinks

You can transform to look like a hyperlink by applying lego-link class.

Google Website 
<a href="http://www.google.com">Google Website</a>

Font Weights

The default font weight for most elements is 400. h2, label, strong, b, and header elements have heavier font weights applied by default. There are also utility classes that can be applied to elements to increase their font weights, lego-medium and lego-bold.

400 font weight: No utility class

500 font weight: lego-medium utility class

700 font weight: lego-bold utility class

 
<p>400 font weight: No utility class</p>
<p class="lego-medium">500 font weight: lego-medium utility class</p>
<p class="lego-bold">700 font weight: lego-bold utility class</p>
