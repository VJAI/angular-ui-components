Text Inputs

Overview

Lego provides both textboxes and textareas to accept textual inputs from users. These controls can be used both with template and reactive forms. They also supports built-in validations and displays customizable error messages.

Textbox

You can create a simple textbox by passing the label and placeholder attributes.

Example

<lego-textbox label="User Name" placeholder="Type User Name"></lego-textbox>

Setting Value

You can set value using either ngModel if you are using classic forms or formControl/formControlName in case of reactive forms. 

<lego-textbox label="User Name" placeholder="Type User Name" [(ngModel)]="userName"></lego-textbox>

Password Textbox

Using the datatype property you can set the type of the textbox. The below example shows how you can create a password textbox by setting the datatype as "password". 

<lego-textbox label="Password" datatype="password"></lego-textbox>

Maxlength

The maxlength attribute is allowed to set the maximum characters allowed by the textbox. 

<lego-textbox label="User Name"
  placeholder="Type User Name"
  maxlength="10">
</lego-textbox>

Disabled Textbox

Adding the disabled attribute will disable the textbox. 

<label="User Name"
  placeholder="Type User Name"
  value="Vijaya Anand"
  disabled>
</lego-textbox>

Validation & Error Messages

Textbox component supports built-in validations like required, minlength, pattern etc. You can also pass custom error messages for the validation errors.

Required

You can make the textbox as mandatory by adding the required attribute. 

<lego-textbox label="User Name"
  placeholder="User Name"
  required
  [(ngModel)]="userName">
</lego-textbox>

Textarea

Simple Textarea

You can create a simple textarea by passing the label and placeholder attributes. 

<lego-textarea
  label="Feedback"
  placeholder="Please give your feedback">
</lego-textarea>

Maxlength

The maximum characters allowed can be restricted using the maxlength attribute. On passing maxlength the control will display a counter at the right bottom. 

<lego-textarea
  label="Comments"
  placeholder="Less than 100 chars..."
  maxlength="100">
</lego-textarea>

Disabled

Setting the disabled attribute will disable the textarea. 

<lego-textarea
  disabled
  label="Comments"
  placeholder="Less than 100 chars..."
  maxlength="100">
</lego-textarea>
